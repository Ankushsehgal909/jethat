Advanced Hero Slider Demo

Experience the power of our Bootstrap 5 enhanced hero slider with smooth transitions, accessibility features, and responsive design.

Key Features

  • Full-width background image slider
  • Smooth fade transitions between slides
  • Auto-slide with manual controls
  • Keyboard navigation support
  • Touch/swipe gestures
  • Accessibility compliant (WCAG 2.1)
  • Responsive design
  • Circular rotating animations

Controls & Navigation

  • Arrow Keys: Navigate slides
  • Spacebar: Play/Pause autoplay
  • Home/End: First/Last slide
  • 1,2,3: Direct slide access
  • Click: Navigation buttons
  • Hover: Pause autoplay
  • Swipe: Touch navigation

Implementation Guide

1. Include Required Files

<!-- CSS Files -->
<link rel="stylesheet" href="styles/main.css">
<link rel="stylesheet" href="styles/hero-slider.css">

<!-- JavaScript Files -->
<script src="scripts/hero-slider.js"></script>

2. HTML Structure

<!-- Load the hero slider component -->
<div id="hero-slider-component"></div>

<script>
// Load component
fetch('components/hero-section/hero-slider.html')
    .then(response => response.text())
    .then(html => {
        document.getElementById('hero-slider-component').innerHTML = html;
    });
</script>

3. Customization Options

// Access the slider instance
const slider = window.heroSlider;

// Update slide content
slider.updateSlideContent(0, {
    title: 'Your Custom Title',
    subtitle: 'Your custom subtitle text',
    badge: '<i class="bi bi-star"></i> Featured'
});

// Update background image
slider.updateSlideBackground(0, 
    'linear-gradient(135deg, rgba(0,0,0,0.7), rgba(220,20,60,0.3)), url("your-image.jpg")'
);

// Listen to slide changes
document.addEventListener('heroSlideChange', (e) => {
    console.log('Current slide:', e.detail.currentSlide);
});

4. Easy Image Updates

To update the background images, simply modify the style attribute in the HTML file:

<div class="hero-slide-bg active" data-slide="0" 
     style="background: linear-gradient(135deg, rgba(0,0,0,0.7), rgba(220,20,60,0.3)), 
            url('assets/images/your-new-image.jpg'); 
            background-size: cover; 
            background-position: center;">
</div>
